Disallow method calls for specified types.

Given parameters: methods - doSomething, classes - MyClass

Noncompliant Code Example

class MyClass {
  int doSomething(int a) { // Noncompliant
    return 42;
  }
}

Compliant Solution

class MyClass {
  int doSomethingOne() { // Compliant
    return 42;
  }
  long doSomethingTwo(int a) { // Compliant
    return 42L;
  }
}
